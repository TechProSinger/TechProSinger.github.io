ProSinger: Technique Controlled Singing Voice Style Conversion

What can ProSinger do?

ProSinger is a controllable singing voice style conversion system that focuses on vocal technique manipulation. Built upon Vevo2 — a unified framework for controllable speech and singing voice generation — it takes original lyrics, a source singing audio, and a phoneme-level technique sequence as input, and outputs a new singing audio that follows the specified technique assignments.

How did we achieve this?

Following the definition of singing techniques in GTSinger, we designed a one-hot phoneme-level technique tokenizer to represent six common singing techniques. And we substitute the original style tokens input in Vevo2's AR model with our technique tokens. Our finetuning process is fully self-supervised. All four tokenizers are freezed. We extended the token embedding dimension of the original AR model to accommodate technique tokens.
